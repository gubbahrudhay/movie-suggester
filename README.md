# Movie Selector App

A smart movie discovery app for people who *can’t decide what to watch*.
It shows trending movies, allows filtering by genre or rating, and even tells you **where to watch**.

---

## Features

* **Trending Movies** (real-time from TMDB)
* **Genre-Based Filtering**
* **Top Rated Filter**
* **Expandable Synopsis**
* **Where to Watch** (Netflix, Prime, etc. via Watchmode)
* **Clean & responsive UI**

---

## Tech Stack

* **Frontend:** HTML,CSS,JS
* **APIs Used:**

  * TMDB (movie discovery & data)
  * Watchmode (streaming availability)
* **Styling:** CSS

---

## APIs Used

### 1. TMDB API

Used for:

* Trending movies
* Genres
* Ratings & posters

https://developer.themoviedb.org/docs/getting-started

---

### 2. Watchmode API

Used for:

* Streaming platforms (Netflix, Prime, etc.)

https://api.watchmode.com/

---

## How It Works

1. Fetch trending movies from TMDB
2. Display them as cards
3. Apply filters (genre / rating)
4. On selecting a movie → fetch streaming info from Watchmode
